Welcome to ROI-master GitHub Pages.

Roim is an advanced image tool to extarct information out of medical images. The main goal of the project is to provide nuclear medicine specialist (both technologists and doctors) with versatile enough instrument to prepare and process results of studies. It could be especially useful for the case when due to some technological or physiological peculiarity acquired data are far from expected thus "standard" clinical applications cannot be used.
Currently it supports images in DICOM format with raw or uncompressed JPEG data only. The programm is written entirely in Java (tm) it runs smoothly on Windows, Linux and (i hope) on other platforms where Java exists.
